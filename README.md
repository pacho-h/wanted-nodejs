# Node.js 엔지니어 과제
## Description
댓글 기능이 있는 익명 게시판 및 키워드 알림 기능 구현

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Docker-compose
API 서비스, Database 컨테이너 실행
- Docker version 27.2.0, build 3ab4256

```bash
$ docker compose up --build -d
```
